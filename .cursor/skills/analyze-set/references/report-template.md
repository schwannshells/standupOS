# Set Analysis Report Template

Use this structure when generating set analysis reports.

---

```markdown
# Set Analysis: [Venue/Event] - [Date]

**Recording**: [filename]
**Duration**: [MM:SS]
**Total laugh time**: [MM:SS] ([percentage]% of set)
**Laughs-per-minute**: [number]
**Average reaction intensity**: [number]/10

---

## Summary

[2-3 sentence overview: how the set went, strongest moments, areas to tighten]

## Transcript

> [MM:SS] Transcript text here...
> [LAUGH 2.3s intensity:7]
> [MM:SS] More transcript text...
> [APPLAUSE 1.5s intensity:8]
> [MM:SS] Continuing...

## Reaction Timeline

| Time | Type | Duration | Intensity |
|------|------|----------|-----------|
| 0:45 | laugh | 2.3s | 7/10 |
| 1:12 | big_laugh | 4.1s | 9/10 |
| ... | ... | ... | ... |

## Dead Zones

| Start | End | Duration | Word Count | Transcript Excerpt |
|-------|-----|----------|------------|-------------------|
| 3:20 | 4:05 | 45s | 87 words | "So the thing about..." |
| ... | ... | ... | ... | ... |

### Tightening Suggestions

**Dead zone at [time range]:**
- Current word count: [N] words over [N] seconds
- The section about [topic] could be tightened by [specific suggestion]
- Consider cutting: "[specific phrases that could be removed]"

## Best Moments

1. **[MM:SS]** - [brief description] (intensity: [N]/10, [duration])
2. **[MM:SS]** - [brief description] (intensity: [N]/10, [duration])
3. **[MM:SS]** - [brief description] (intensity: [N]/10, [duration])
```
